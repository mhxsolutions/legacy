Auto Coverage ($1,000,000) "additionally insured" Check the box if you have a copy of the insurance certificate showing auto coverage with Budway listed as additionally insured

Note: A statement saying that "additional insured endorsement has been requested" is NOT adequate--we need the "additionally insured" endorsement to be actually given
Enter the amount of the coverage in the Notes field; if the additionally insured endorsement is not provided, add "not add'l insured" in this field and DO NOT CHECK THE BOX until the endorsement is received
